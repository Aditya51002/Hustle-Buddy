# 📚 Study Buddy App

> A modern Android productivity app built for students to manage study schedules, tasks, focus sessions, quizzes, reminders, and progress — all in one place.

## ✨ Overview

**Study Buddy App** is a frontend-only Android application developed using **Kotlin** and **Jetpack Compose**.  
It is designed to help students stay organized, focused, and consistent in their learning journey.

Instead of switching between multiple apps for planning, tasks, focus timers, and quizzes, Study Buddy App brings everything together into one clean and student-friendly experience.

## 🚀 Features

## 🎯 Project Objective

The goal of this project is to create a **student-focused study management app** that improves productivity and makes academic planning easier.

This project also demonstrates:
- Modern Android UI development
- Multi-screen app navigation
- Reusable Compose components
- Scalable project structure
- Frontend architecture ready for backend integration
